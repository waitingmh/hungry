### DEMO
展示数据通过mock.js,基于react框架(create-react-app+项目名)

### `npm install` (下依赖)

### serve with hot reload at localhost:8080
### `npm start` (起服务)

### `npm run build` （构建生成环境）

### 目录结构

<pre>
├── pulic                     // 静态资源相关
├── src                        // 源代码
│   ├── font                   // 引入icon
│   ├── mock                   // mock数据
│   ├── router                 // 路由
│   ├── store                  // 全局store管理
│   ├── view                 // 视图
│   ├── index.js               // 入口 初始化
│   └── index.scss                // 全局样式
├── .gitignore                 // 用于Git配置不需要加入版本管理的文件
└── package.json               // 项目依赖管理
</pre>

### 技术栈

* axios
* vue-router
